Perguntas Frequentes

O que é Big Data?

+

Big Data refere-se a conjuntos de dados extremamente grandes e complexos que requerem ferramentas especializadas para coleta, armazenamento, processamento e análise. É caracterizado pelos 5 Vs: Volume, Velocidade, Variedade, Valor e Veracidade.

Qual a diferença entre Big Data e análise de dados tradicional?

+

A análise de dados tradicional trabalha com volumes menores e estruturados, enquanto Big Data lida com volumes massivos, dados não estruturados e em tempo real, exigindo tecnologias específicas como Hadoop e Spark.

Como o Big Data é usado na política?

+

Na política, o Big Data é usado para análise de opinião pública, segmentação de eleitores, otimização de campanhas eleitorais, monitoramento de redes sociais e formulação de políticas públicas baseadas em evidências.

Quais são as principais ferramentas de Big Data?

+

As principais ferramentas incluem: Hadoop, Apache Spark, Python (Pandas, NumPy), R, SQL, NoSQL (MongoDB, Cassandra), Tableau, Power BI, e plataformas cloud como AWS, Google Cloud e Azure.

Quais são os riscos do Big Data?

+

Os principais riscos incluem violação de privacidade, discriminação algorítmica, segurança de dados, manipulação de informações, viés nos algoritmos e dependência excessiva de tecnologia.

Como começar uma carreira em Big Data?

+

Para começar, é recomendado aprender programação (Python/R), estatística, SQL, ferramentas de visualização, e obter certificações em plataformas cloud. Experiência prática com projetos é fundamental.
